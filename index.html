Couscous Academy

Couscous Academy is an online educational platform offering video courses and traditional couscous recipes, designed to teach the tunisian Coscous
Technologies: Laravel · Vue.js · MySQL · Tailwind CSS

Category
Web Development
Client
Couscous Academy association
Start Date
Feb 20, 2025
Developed By
Mohamed Amine Mrissa

Project Description

Couscous Academy is a full-featured e-learning platform. The platform provides structured video lessons, detailed traditional recipes, and educational content. Users can browse and follow cooking tutorials, explore regional culinary traditions, and engage with expert chefs. I developed both the frontend and backend of the platform using Angular for user interface and Spring Boot for backend. Data is stored and managed via MySQL.

A unique feature of the platform is its restaurant profile system, where local restaurants can promote their services, showcase their specialties. Clients can browse these profiles, interact by posting ratings and reviews.

An admin dashboard was also implemented, allowing content managers to upload and organize videos, manage user accounts, moderate recipe submissions, and track platform activity.
